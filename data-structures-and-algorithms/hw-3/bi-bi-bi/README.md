## Bi Bi Bi ##
You are given an array of N numbers and afterwards you are given Q queries with a number P. 
For each of the queries you have to answer which is the index of the first meeting of P and the number of the last meeting of P within the array in a sorted state.
If P is not into the array you must return -1.
### Input format ###
Number N, followed by N numbers - the numbers of the array. Follows a number Q - number of the queries. For each number you are given a number P.
